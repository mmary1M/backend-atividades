

# Trabalho desenvolvimento de sistemas

Neste projeto terremos como iniciativa e objetivo fazer um backend para armazenar dados sobre animes da sua preferencia. Esse projeto usa como base o prisma e tem varias bibliotecas auxiliando, entre elas: http, cors(que bloqueia caso a pessoa tentar entrar com coisa errada), express e entre outras.
